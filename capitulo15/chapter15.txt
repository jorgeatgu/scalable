# Novedades SVG 2

En febrero de 2015 el SVG Working Group se reunió durante cuatro días en Sidney. Desde aquella reunión el repositorio de SVG alojado en GitHub echa humo. Así que en este capítulo voy a tratar la serie de cambios(no todos) que va a suponer SVG 2.

Los nuevos elementos ya no utilizarán la nomenclatura *camelCase*, así que nuevos elementos como *meshGradient* se llamará *mesh*. Queda pendiente saber que pasará por ejemplo con *linearGradient*.

Vamos a poder añadir texto alrededor de las formas básicas. Aquellos que trabajeis con programas como Illustrator, o hayais tenido la suerte de trabajar **con el mejor programa de gráficos vectoriales como Freehand** ya os sonará esto de añadir texto alrededor de formas básicas.


Added the ability to use 'auto' for the ‘width’ and ‘height’ attributes on ‘image’.

Added ‘lang’ attribute on ‘desc’ and ‘title’ elements.

Removed the baseProfile and version attributes from the ‘svg’ element.

Added new B and b "bearing" path commands.
Added new R and r Catmull-Rom curve commands.
